# StudyMate AI

StudyMate AI is an educational web application built with Streamlit.

## Features

- AI Notes Generator
- MCQ Test System
- Past Papers Library
- Admin Dashboard
- Exam Countdown
- Progress Analytics

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
